## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript를 이용한 서버페이징 서버

## Installation

```bash
$ npm install
```

또는 `yarn`을 이용할 경우

```bash
$ yarn
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

또는 `yarn`을 이용할 경우

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```
